# Webcam_Access
Someone is watching without you knowing!?
</br>
CySmash is a browser-based mini-game that combines interactive gameplay with cybersecurity awareness, demonstrating how users can unknowingly expose sensitive data by granting permissions.</br></br>
**Features**</br>
- Word-based game (identify and click threats)
- Score, level progression, and timer
- Start/Stop controls
- Automated webcam recording (after permission) 
- 10-second video segmentation
- Local storage using IndexedDB
- Cloud upload using Cloudinary
- Security awareness popup with device info</br>
</br>**Security Insight**
- Modern browsers require user permission for camera access.
- The real risk is social engineering, where users click “Allow” without verifying the source.</br>
</br> **Device Information Collected**
- Browser (user agent)
- Platform / OS
- Timezone
- Screen resolution
- Language
</br></br>
**Technologies Used**
- HTML, CSS, JavaScript
- MediaRecorder API
- IndexedDB & localStorage
- Cloudinary (cloud storage)
- GitHub Pages 
</br></br>
**Conclusion** </br>

CySmash demonstrates that security risks often come from user actions, not system flaws, highlighting the importance of cautious permission handling.
